git pull origin main 


(subir datos)
git add .

git push origin main


(Para listar todos los commits)
gitl es un alias personalizado de: 

git log --all --decorate --oneline --graph


(Alias personalizado)
git config --global alias.l "log --all --decorate --oneline --graph"


(Metodo mas comodo?)

git config pull.rebase false

¿Cuándo usar esto?
Si prefieres un historial con los merges explícitos (útil para ver claramente cuándo se combinaron ramas).

Configura Git para que no use rebase al hacer git pull, sino que utilice el método por defecto: merge.
